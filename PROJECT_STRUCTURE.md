# Структура проекта

## Новая модульная структура

```
artificial-life/
├── src/                          # Исходный код
│   ├── config/                   # Конфигурация
│   │   └── constants.js         # Константы игры
│   ├── worlds/                   # Миры
│   │   ├── World.js             # Базовый класс мира
│   │   ├── DefaultWorld.js      # Мир по умолчанию
│   │   └── index.js             # Экспорт миров
│   ├── items/                    # Предметы
│   │   ├── tools/               # Инструменты
│   │   ├── clothes/             # Одежда
│   │   ├── food/                # Еда
│   │   ├── resources/           # Ресурсы
│   │   ├── ItemRegistry.js      # Реестр всех предметов
│   │   └── index.js             # Экспорт
│   ├── agents/                   # Агенты
│   │   ├── Agent.js             # Базовый класс агента
│   │   ├── AgentTypes.js        # Типы агентов
│   │   ├── AgentsManager.js      # Менеджер агентов
│   │   └── index.js             # Экспорт
│   ├── ui/                       # Интерфейс
│   │   ├── panels/              # Панели
│   │   ├── modals/              # Модальные окна
│   │   └── components/           # Компоненты
│   ├── network/                  # Сетевое взаимодействие
│   │   ├── NetworkManager.js    # Менеджер сети
│   │   └── index.js             # Экспорт
│   └── utils/                    # Утилиты
│       ├── logger.js            # Логирование
│       └── helpers.js           # Вспомогательные функции
├── backend/                      # Серверная часть
│   └── server.js                # Сервер
├── index.html                    # Главная страница
├── style.css                     # Стили
└── main.js                       # Точка входа (будет обновлен)
```

## Преимущества новой структуры

1. **Модульность**: Каждый компонент в своем каталоге
2. **Расширяемость**: Легко добавлять новые миры, предметы, агентов
3. **Организация**: Логическое разделение по функциональности
4. **Масштабируемость**: Проще поддерживать большой проект

## Как добавить новый мир

1. Создайте файл `src/worlds/NewWorld.js`
2. Наследуйтесь от `World`
3. Реализуйте методы `generateTerrain()`, `draw()`
4. Добавьте экспорт в `src/worlds/index.js`

## Как добавить новый предмет

1. Добавьте определение в `src/items/ItemRegistry.js`
2. Разместите в соответствующей категории (tools, clothes, food, resources)
3. Используйте через `getItem(itemId)`

## Миграция

Старые файлы (`world.js`, `agents.js`) будут постепенно перенесены в новую структуру.
